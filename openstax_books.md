# OpenStax Textbooks in CNXML Git Repositories

This is a complete list of English-language OpenStax textbooks stored as CNXML Git repositories, covering all educational levels (high school, undergraduate, AP, etc.). Bundles are expanded to show individual books, and standalone books are noted. The list includes titles like Microbiology, Chemistry titles (Chemistry 2e, Chemistry: Atoms First 2e, Organic Chemistry), and Astronomy (covering astrophysics topics, as no separate astrophysics book exists). All repositories are hosted under the 'openstax' or 'cnx-user-books' organizations on GitHub.

- **Prealgebra Bundle** (contains Prealgebra 2e, Elementary Algebra 2e, Intermediate Algebra 2e): [https://github.com/openstax/osbooks-prealgebra-bundle](https://github.com/openstax/osbooks-prealgebra-bundle)
