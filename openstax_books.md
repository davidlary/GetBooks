# OpenStax Textbooks in CNXML Git Repositories

This is the authoritative whitelist of English-language OpenStax textbooks stored as
CNXML Git repositories, covering all educational levels (high school, undergraduate,
AP, etc.). `get_books.py` uses this file to filter GitHub API results — only repos
listed here are cloned. All repositories are hosted under the `openstax` or
`cnx-user-books` organizations on GitHub.
